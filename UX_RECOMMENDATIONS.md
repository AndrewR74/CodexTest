# Session Tile Discovery UX Recommendations

1. Add a text search box that matches session name, speaker names, code, and location.
2. Support multi-select category filters with visible chips and a "Clear all" action.
3. Add sort controls (time, popularity, availability, featured).
4. Use lazy loading / pagination for large day groups to keep scroll fast with 100s of sessions.
5. Highlight sessions that conflict with already-selected sessions in the attendee's agenda.
6. Add a compact/list toggle to let users scan quickly or read details.
